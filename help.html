Draw My Thing Cheat Help


How to use this

  • 1. Select the length of the word you need to guess with the buttons
  • 2. Enter the hints in the textboxes as they appear in the game
  • 3. Words that match the hints and word length will appear in the word list
  • 4. Try all the words in the list and you'll end up guessing what it is
  • 5. If you need to guess multiple words, click on the + button then find
  • the word in the word list (giving hints would have been slower in this case)
  • 6.Not all the words in the game are known, so the word might not be in the list
